# Documentos de Referência - EletriLab

Esta pasta destina-se a armazenar **documentos de referência** (PDF e Excel) para apoio ao treinamento, aprendizado e automação do sistema EletriLab.

## Formato Esperado

### PDFs
- Manuais de equipamentos (Megger, microhmímetro, hipot)
- Normas técnicas (NBR 5410, IEC 60364)
- Procedimentos de ensaio
- Modelos de relatórios

### Planilhas Excel (.xlsx)
- Tabelas de referência (ampacidade, bitolas)
- Exemplos de relatórios preenchidos
- Dados para treinamento e validação

## Uso Planejado

- **Treinamento:** Materiais para capacitação de usuários
- **Automação:** Dados de referência para validação e sugestões
- **Templates:** Modelos de relatórios para importação futura

## Como Adicionar

1. Coloque os arquivos PDF ou XLSX nesta pasta
2. Use nomes descritivos (ex: `megger-manual.pdf`, `tabela-ampacidade.xlsx`)
3. O sistema poderá consumir esses arquivos em versões futuras

## Estrutura Sugerida

```
documentos/referencias/
├── README.md
├── normas/           # PDFs de normas
├── manuais/          # Manuais de equipamentos
├── modelos/          # Modelos de relatório Excel
└── dados/            # Dados para treinamento
```
