# Changelog

Todas as mudanças notáveis deste projeto serão documentadas aqui.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/),
e este projeto segue [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-04-27

### Added
- Geração de relatórios (Megger/IR, Microhmímetro, Hipot, Cabo e Disjuntor)
- Dois modos de entrada: manual e cálculo automático
- Exportação em PDF e Excel
- Persistência local (IndexedDB via Dexie) e sincronização opcional (Firebase)
